# Личный кабинет студента

## Описание
Проект представляет собой веб-приложение, предназначенное для студентов и преподавателей. Основная цель — централизованный доступ к учебной информации.

## Основной функционал
- Просмотр расписания занятий  
- Просмотр и проверка успеваемости (оценок)  
- Загрузка учебных материалов и заданий  
- Отправка выполненных работ  
- Обратная связь с преподавателями

## Технологии
- Язык: C#  
- Фреймворк: ASP.NET Core MVC  
- Система контроля версий: Git  
- Платформа: GitHub
